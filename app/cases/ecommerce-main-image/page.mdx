
# 电商主图：把产品“放进”更卖货的场景

我们示范如何利用 Nano Banana 生成合规、统一风格的电商主图：

1. 上传白底产品图；  
2. 提示词指定目标场景（仓库、农田、展台）；  
3. 保持产品比例、角度与结构合理；  
4. 保留水印与 AI 标识，避免“真实照片”误导。

> 实战案例：农业机械（拖拉机、喷雾器、播种机）主图构图清单附在下载包中。
